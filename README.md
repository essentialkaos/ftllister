`FTLLister` (_faster-than-light lister_) is simple utility for listing directories with huge amounts of files (several millions).

### Build & Install

Build requires `make` and `gcc` or `clang`.

```bash
make
sudo make install
```

### Usage

```
ftllister /some/path
```
