### FTLLister

`FTLLister` (_faster-than-light lister_) is utility for listing directories with huge amounts of files (several millions).

#### Build & Install

Build requires `gcc` and `make` utilities.

```
make
make install
```

#### Usage

```
fls /some/path
```
